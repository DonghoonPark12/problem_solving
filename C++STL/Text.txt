
# 6장
시퀀스 컨테이너는 저장 원소가 삽입 순서에 따라 위치가 정해진다.
vector, list, deque
(※ vector와 dequq은 배열 기반 컨테이너지만, list는 노드기반 컨테이너 이다.)

# 7장
연관 컨테이너는 특정 정렬 규칙에 의해 저장된 원소가 정렬된다.
set, map, multiset, multimap
모든 연관 컨테이너는 노드 기반 컨테이너. 균형 이진 트리로 구현되어 있다. 


# 11장
컨테이너 어댑터는 다른 컨테이너 인터페이스를 변경한 컨테이너.
stack, <-- deque
queue, 
priority_queue <-- vector
3가지 가 있다.
